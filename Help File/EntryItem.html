How to process the items data

After you have successfully logged in to the system, you can enter the required data.

First of all, to enter the item data, click the Entry menu from the menu bar. The item form will be appeared.

When it is come out, just type the new item name in the item name textbox. And choose the correct type of category in the combo box.

After that the enter the neccessary description for the particular items.

After that, to save the typed data , click save button or Alt+S. And the following message will appeared.

If the current data need to be edited, double click on the grid line and click the "update" button or Alt+U and edit the required data.

Then, the following message will be appeared.

In order to delete the data from the category form, double click on the Grid line first and click "Delete" button or Alt+D.

The message will be appeared.
