How to process the category data

After you have successfully logged in to the system, you can enter the required data.

First of all, to enter the category data, click the Entry menu from the menu bar. The category form will be appeared.

When it is come out, just type the new category name in the category name textbox.

After that, to save the AuxilaryMaterial as a category, click save button or Alt+S. And the following message will appeared.

If the current data need to be edited, double click on the grid line and click the "update" button or Alt+U and edit the required data.

Then, the following message will be appeared.

In order to delete the data from the category form, double click on the Grid line first and click "Delete" button or Alt+D.

The message will be appeared.
