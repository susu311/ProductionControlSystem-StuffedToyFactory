How to process Search Menu

This menu is for searching or retrieving data from the system to be easier and more time-saving. In this menu, there are altogether twelve searching form to access and find out data from various aspects. The search forms are:

 

1. Search for Item information

2. Search for Customer Information

3. Search for Employees by their status

4. Search for orders

5. Search for jobs

6. Search for Purchases

7. Search for Cutting Information

8. Search for Sewing Information

9. Search for Stuffing Information

10. Search for Packaging Information

11. Search for Monthly OrderReceipt

12. Search for Annually OrderReceipt
